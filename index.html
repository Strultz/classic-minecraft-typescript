This game requires JavaScript and WebGL. Press escape to release the mouse.

Controls:

WASD to move
Space to jump
R to respawn above the level
G to add a mob
Y to swap mouse Y axis
1-5 to change building block type
N to generate a new level
Left mouse button to remove or add a block
Right mouse button to switch editing mode
