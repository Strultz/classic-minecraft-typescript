This game requires Java. Press escape to release the mouse.
Please read this if you're having trouble loading/saving maps.

Controls:

WASD to move
Space to jump
R to respawn above the level
G to add a mob
Y to swap mouse Y axis
F to toggle rendering distance
Escape to release mouse and open game menu
1-4 to change building block type
Left mouse button to remove or add a block
Right mouse button to switch editing mode
